To illustrate how the Commander Keen game engine works, several screenshots, images, sprites, and textures—belonging to and copyrighted by id Software and by \textit{Game Engine Black Book: Wolfenstein 3D} by Fabien Sanglard—are reproduced in this book.\\

\par
The following items are used under the “fair use” doctrine from id Software:
\begin{enumerate}
	\item All in-game screenshots, title screen and retail cover.
	\item All in-game sound menu screenshots.
	\item All 2D tile textures (background and foreground tiles).
	\item All 2D picture assets.
    \item All 2D sequence sprites (Commander Keen, BROCCOLASH, CARROT, MELONLIPS, TOMATOOTH).
    \item All TED5 editor screenshots.
\end{enumerate}
