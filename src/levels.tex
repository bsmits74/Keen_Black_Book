\par
Level 6, 8, 11 and 13 are secret levels in Keen Dreams that cannot be reached from the map. They can only be accessed using the cheat code \cw{F10+W} (warp). It is unknown whether or not these incomplete levels were intentional, development of Keen Dreams was rushed, or because of some other problem. It is also worthy of note that all these levels have been removed from the original registered versions.\\
